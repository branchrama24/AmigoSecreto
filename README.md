Amigo Secreto
Juego de Amigo Secreto donde los usuarios pueden agregar sus nombres y hacer un sorteo para asignar a cada persona un amigo secreto.

Funcionalidad
Agregar nombres a la lista.
Sortear un amigo secreto para cada participante.
Ver resultados del sorteo sin repetición de asignaciones.
Cómo Usarlo
Agregar nombres: Escribe el nombre de un amigo y presiona "Añadir".
Sortear: Haz clic en "Sortear amigo" para asignar un amigo secreto a cada participante.
Ver resultados: Los resultados se mostrarán debajo del botón de sorteo.
Requisitos
Navegador web moderno.
